XKCD URI OSX

An extremely simple OS X app which opens URIs that use the xkcd scheme in your default browser.

What?

Something like xkcd://201. Apparently that's a thing.

Why?

The hell if I know. Some iOS apps for reading XKCD (don't even get me started on that) started using it so the comics would open in their app. They're showing up in the wild a bit, and this app just removes the annoyance when you encounter one.

How?

XKCD URI OS X just tells OS X that it knows how to open that type of URI. When one is clicked the app figures out the real URL for the comic and opens it in your default browser.

Installation Instructions

  1. Download the latest version from Github.
  2. Unzip the archive.
  3. Move XKCD to the Applications folder.
  4. Right-click on XKCD and select "Open".
  5. When prompted, click the "Open" button.
  6. After a few seconds, XKCD #353 (Python) will open in your default browser.

Using

Just leave it be. If it's doing its job correctly you'll forget it's even there.
